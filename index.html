Perangkat Akhir

Penjelasan

Perangkat akhir adalah perangkat keras yang mendukung web yang berfungsi sebagai sumber atau tujuan data yang ditransfer melalui jaringan. Misalnya, workstation, laptop, komputer desktop, printer, pemindai, tablet, atau ponsel. Perangkat akhir memainkan peran antarmuka antara pengguna akhir dan jaringan komunikasi.

Berdasarkan klasifikasinya, perangkat akhir dibagi menjadi klien dan server. Klien adalah perangkat akhir yang memiliki perangkat lunak (seperti browser web) yang diinstal untuk memungkinkan permintaan data dari server dan menampilkan data yang diterima.

Di sisi lain, server adalah perangkat akhir yang dilengkapi dengan perangkat lunak yang memungkinkannya menyediakan informasi dan layanan (misalnya, email atau halaman web) ke perangkat akhir lainnya.

Perbedaan antara perangkat akhir dimungkinkan berdasarkan alamat IP-nya. Ketika perangkat akhir tertentu akan memulai komunikasi, alamat IP perangkat akhir tujuan menunjukkan ke mana pesan akan dikirimkan.

Ada banyak contoh perangkat pengguna akhir, termasuk komputer desktop, laptop, printer, tablet, dan server.
